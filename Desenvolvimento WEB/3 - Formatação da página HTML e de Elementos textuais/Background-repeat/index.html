no-repeat

A imagem não se repete


repeat

A imagem é repetida até cobrir todo o fundo. A última imagem será cortada se não couber.



repeat-x

imagem se repete na horizontal (eixo x)


repeat-y

A imagem se repete na vertical (eixo y)


space

A imagem se repete sem recortes e com distribuição de espaçoes em branco entre elas para que não haja recortes da imagem


round

A imagem se esticará ou comprimirá para se adaptar a tela.
