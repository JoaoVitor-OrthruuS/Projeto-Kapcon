Background-attachment


A propriedade background-attachment define se uma imagem de fundo é fixa ou rola com o resto da página. Os valores aplicáveis a esta propriedade são:
fixed: o fundo fica sempre fixo e não rola quando o usuário utliza a barra de rolagem da página;
scroll: o fundo sempre rola quando o usuário utliza a barra de rolagem da página.
